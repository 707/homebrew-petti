# 707 Petti

## How do I install these formulae?

`brew install 707/petti/<formula>`

Or `brew tap 707/petti` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "707/petti"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
